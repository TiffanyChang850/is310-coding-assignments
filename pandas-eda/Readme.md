# Top 500 Novels EDA #

## Overview ##

This project explores the Top 500 Novels dataset and New York Times Bestseller dataset using exploratory data analysis (EDA). The goal is to identify patterns in genre, publication year, ratings, and author representation.

## Files ##

* `PronounAndGenrePatternsInTopNovels.ipynb` – Main Jupyter Notebook with analysis and visualizations

## Data Sources ##

* Top 500 Novels dataset
  https://raw.githubusercontent.com/melaniewalsh/responsible-datasets-in-context/main/datasets/top-500-novels/final_merged_dataset_no_full_text.tsv

* NYT Bestsellers dataset
  https://raw.githubusercontent.com/ecds/post45-datasets/main/nyt_full.tsv

## Methods ##

* Data cleaning and merging using Pandas
* Exploratory Data Analysis (EDA)
* Data visualization using Altair

## Key Findings ##

* Some genres are more represented than others
* Publication trends vary across time
* Ratings are relatively consistent across genres
* Author representation shows possible imbalances

## How to Run ##

1. Open the notebook in Jupyter Notebook or VS Code
2. Run all cells in order
3. Make sure you have the required libraries installed:

   * pandas
   * altair
