## SWAPI Europeana ##

The Star Wars API (SWAPI) is a free, public web API that provides structured data about the Star Wars universe. It includes information on characters, films, planets, species, vehicles, and starships. Each resource is accessible through RESTful endpoints and returns data in JSON format, making it easy to retrieve and use in programs. The API also supports searching and linking between resources, allowing users to navigate related data such as a character’s homeworld or the films they appear in.

I selected SWAPI because it is simple to use and does not require authentication, which makes it ideal for practicing API requests. The data is well organized and easy to understand, allowing me to quickly retrieve a specific item and then search for related content in the Europeana collection. This made it a good choice for combining data from two APIs and storing the results.
